# MAGIC WHEELS 根目录导航

整理时间：2026-05-19

这个根目录层是当前项目的工作视图，方便直接找资料；原来的 `需求整理`、`资料`、`设计稿` 仍然保留，作为源库和历史目录。

## 根目录怎么用

### 01-需求与PRD

放客户需求、PRD、回填表、早期网站表。

主要文件：

- `客户需求文档-MAGIC-WHEELS.md`
- `MAGIC-WHEELS-全站PRD.md`
- `MAGIC-WHEELS-建站资料回填表-Shirley 0518.xlsx`
- `MAGIC-WHEELS-建站资料回填表.xlsx`
- `B端网站.xlsx`

### 02-产品资料

放按 SKU 归类的产品图、手册、字体、产品映射说明。

主要内容：

- `产品图`
- `手册资料`
- `产品SKU资料与素材映射.md`

### 03-工厂与合规

放工厂图、证书、测试报告、案例资料。

主要内容：

- `工厂素材`
- `证书合规`
- `案例素材`

### 04-网站UI

放全套 PC / 手机 UI 图、索引图、UI 清单。

### 05-上线配置

放联系资料、上线配置、询盘页真实信息相关文档。

### 06-企业微信待下载

放企业微信待下载入口和下载路径清单。

### 07-表格完整导出

放三个 Excel 的完整导出，以及新版回填表里的嵌入图片抽取结果。

主要内容：

- `完整表格导出`
- `回填表嵌入图片`
- `企业微信截图`

### 08-项目管理

放总台账、企业微信路径清单、遗漏审计报告、后续执行说明。

主要文件：

- `MAGIC-WHEELS-项目资料总台账.md`
- `MAGIC-WHEELS-WordPress临时部署计划.md`
- `MAGIC-WHEELS-企业微信下载路径清单.md`
- `MAGIC-WHEELS-资料整理遗漏审计报告.md`
- `表格信息记忆说明.md`

### 09-WordPress建站

放 WordPress 本地开发环境、定制主题、内容模型、导入脚本和 Railway 临时部署说明。

主要内容：

- `docker-compose.yml`
- `README.md`
- `wp-content/themes/magic-wheels`
- `wp-content/mu-plugins/magic-wheels-core.php`
- `data/products.json`
- `tools/install-local.ps1`
- `railway/README.md`
