## 表格中的客户问题

客户问：“需要什么样信息体现？最好给个示例”

建议回答：

案例至少需要这些信息：

| 字段 | 示例 |
| --- | --- |
| 客户类型 | European Distributor / North American Retail Chain / Private Label Brand |
| 市场 | US / EU / LATAM / Middle East |
| 产品系列 | 3-Wheel Toddler Scooters / 2-Wheel Kids Scooters / Electric Scooters |
| 客户痛点 | 旺季交期、合规文件、包装统一、定制、MOQ、上新节奏 |
| 解决方案 | 产品组合建议、样品确认、包装定制、测试报告支持、生产排期 |
| 结果 | Helped the buyer prepare a retail-ready scooter range for seasonal launch |
| 可公开图片 | 产品白底图、包装图、发货图、展厅图、工厂图 |
| 是否脱敏 | yes / no |

## 网站使用方式

- Case Studies 页面可先用匿名案例，不写真实客户品牌。
- 结果描述保守，不硬写销售数据。
- 如无真实案例图片，可用产品图、包装图、发货图、工厂图作为辅助。

## 仍需补充

- 真实案例图片。
- 客户类型和市场确认。
- 可公开的结果描述。
- 是否允许展示客户行业或地区。
