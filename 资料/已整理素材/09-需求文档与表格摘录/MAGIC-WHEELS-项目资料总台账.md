## 11. 回填表里的两个问题建议答法

### 产品图片素材是否与产品树要求相同？

是，同一套素材要求。区别是：

- 产品树负责 SKU 级资料主入口。
- 产品图片素材表负责把图片拆成主图、细节图、包装图、场景图、视频截图等用途。

### 案例素材需要什么信息？

建议至少提供：

- 客户类型：如零售商、进口商、经销商、品牌方。
- 市场：如 US / EU / LATAM。
- 产品系列：如 3-Wheel Scooter / 2-Wheel Scooter。
- 客户痛点：交期、合规、包装、定制、MOQ、上新节奏。
- 解决方案：产品组合、打样、包装、检测、交付节奏。
- 结果：可保守表达，不一定写具体销售数据。
- 可公开图片：产品、包装、发货、展厅、工厂。
- 是否需脱敏：客户名、订单号、品牌 Logo、证书编号。

示例：

| 字段 | 示例 |
| --- | --- |
| 客户类型 | European Distributor |
| 市场 | EU |
| 产品系列 | 3-Wheel Toddler Scooters |
| 痛点 | 需要儿童滑板车新品组合，同时兼顾 EN71 资料和包装统一性 |
| 解决方案 | 推荐 3 款主推型号，提供颜色与包装定制，配合测试报告打码截图 |
| 结果 | Helped the buyer prepare a retail-ready scooter range for seasonal launch |
| 图片 | 产品白底图、包装图、出货图 |
| 脱敏 | yes |
