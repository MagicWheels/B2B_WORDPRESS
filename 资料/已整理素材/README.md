# 已整理素材目录说明

这个目录现在已经放入现有可归档素材，采用“复制归档，不移动原件”的方式整理。

## 子目录规则

- `00-索引`：缩略图索引、总索引、素材整理说明。
- `01-产品图`：按 SKU / 系列归档的产品主图、细节图、场景图。
- `02-工厂素材`：工厂外观、办公室、后续企微工厂图入口。
- `03-视频素材`：视频需求说明和后续企微视频入口。
- `04-证书合规`：证书 Logo、测试报告线索、展示规则。
- `05-案例素材`：案例素材模板、现有案例线索。
- `06-联系资料`：上线联系信息、表单配置、二维码素材入口。
- `07-品牌字体`：字体文件。
- `08-手册资料`：客户手册 PDF / AI / 提取报告。
- `09-需求文档与表格摘录`：需求文档、PRD、回填表、企业微信路径清单等项目资料副本。
- `10-回填表嵌入图片`：从新版 Excel 中抽出的 23 张嵌入图片，包括产品图、包装/PDQ、证书 Logo、测试报告截图、案例流程图。

## 命名规则

```text
SKU-用途-序号.扩展名
factory-模块-序号.扩展名
certificate-名称-用途.扩展名
case-市场-客户类型-序号.扩展名
```

示例：

```text
MWKE082-main-01.png
MWKE082-detail-wheel-01.jpg
factory-qc-01.jpg
certificate-bsci-logo.png
case-eu-distributor-01.jpg
```

## 当前整理口径

现有 `资料\Links` 里的图片已按文件名和缩略图内容先行归类；新版回填表里的嵌入图片也已单独抽取并复制到对应分类。由于部分图片没有原始 SKU 标注，归档中保留了 `待确认` 类目录，后续拿到企业微信原始素材后再二次校准。
